Projeto desenvolvido durante o Curso de HTML5 e CSS3 do Gustavo Guanabara (Curso em Vídeo).
O objetivo é criar uma interface simples que simula a tela de um smartphone exibindo diferentes páginas de redes sociais dentro de um iframe.
A página exibe um celular ao centro, com botões laterais representando redes sociais. Ao clicar, o conteúdo da respectiva página aparece dentro da tela e ao clicar em acessar você é direcionado para ela.
